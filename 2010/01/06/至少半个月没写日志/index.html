至少半个月没写日志

发表于 2010-01-06 | 更新于: 2014-09-01

难道这个也要算改变。

现在我正在图书馆的一楼，坐在一个多安逸的椅子上，还可以左右摇摆。

老实说图书馆学习的氛围很好，哪像我们那寝室，要是看得进去书，我就不是我了…

前天我第一次进图书馆是为了学习，复习功课。终于体会到了期末好多了人了，来迟了肯定是莫得位置的，说明我们成大的学生还是多爱学习的嘛。

昨天和今天，都是在一楼坐到的，虽然一楼不是那么安静，但是这个椅子坐到那是相当舒服啊…安逸，这两天一直在看C语言，其实很慢，我想今天应该查不到可以看完了哦，也许嘛，我这是对她这么说的。

我太容易受外界的影响了，比如说美女啊，比如说某一瞬间的想法啊，比如说胡思乱想啊。这就使我浮躁了，只要一浮躁当然啥子就都看不进去了哦。

但是不得不肯定这两天是相当有收获的，明天继续数学，今天晚上继续12点睡觉。我其实怀念以前早睡的日子，但是我更喜欢有人关心有人问候的日子。权衡一下…很明显可以发现哪个好。

哦，似乎还忘了一个事，昨天才发现用C语言编一个九九乘法表其实很简单，莫得我们想得那么难。

#include ”stdio.h”    //九九乘法表。

void main()

{

    int i,j,s;

    for (i=1;i<=9;i++)

     {

     for(j=1;j<=i;j++)

         {  s=i*j;

       printf (“%d*%d=%-4d”,i,j,s);

         }

         printf (“\n”) ;

     }

   

} 

昨天，中午在图书馆过了，不要以为我实在学习，我其实想，但是我太懒了，就在图书馆耍了一中午。

哦，说到这，又想到下午和美国人聊天，居然他最讨厌的是中国和俄罗斯（或者是苏联），这个也不出乎我们意料，但是听到他说这个我还是很不舒服的。他说到了一个事—“天安门”，不要问我太多，这个东西很严重。点到为止。我一直在说我们需要时间，我们在改变。他说他不相信中国政府，他说美国政府不管做什么好事或者坏事都是公布出来了的。事实上真的是这样？我想也不是吧…还可以想见那些外国媒体把我们中国想成什么样了。鄙视那些瞧不起我们的人，虽然我们也有不好的地方。

哎，纳闷扯到一个很有高度的问题了，我不想和政治沾边。我只是一个中国人，我爱中国。

有人说，男生不应该写日志，我不这样认为，我只是把我想说的写下来，我怕这些东西不久就从我脑袋消失了…

杂记—朋友们啊，多来看看哈。
