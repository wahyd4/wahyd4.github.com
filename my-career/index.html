Libr Mobile

是Libr升级后的客户端，主要功能是帮助用户推荐基于地理位置附近流行的图书，和帮助用户找到和他兴趣相似的用户。本质是一个图书社交的应用，已发布到Apple store。
App Store 下载地址：https://itunes.apple.com/cn/app/libr/id865672150?l=zh&ls=1&mt=8
Github项目地址：https://github.com/wahyd4/Libr-mobile
网站：http://librme.com/

Libr

是一个用户分享、借阅、管理、展示图书的小应用，目前正在为Thoughtworks Chengdu 办公室服务。我们正不断地为更好的方便同事借书而升级改造Libr。后期我们希望Libr可以为公开为所有用户服务。
Github项目地址：https://github.com/wahyd4/Libr
网站： http://librme.com/

Sohub

Sohub是一个大屏幕信息集成、展示、分享平台，网页争对大屏幕进行优化，目前可以显示微博、邮件信息、微信上传普通文字信息、图片等信息，我是希望可以还可以争对企业做相关内容的展示平台，可成为企业对外展示的工具、平台。
Github项目地址：https://github.com/wahyd4/sohub
网站： http://sohub.toozhao.com

PM2.5 Alfred Workflow

这是一个用于查询中国很多城市的实时PM 2.5值的一个Alfred Workflow。不少城市拥有多个监测点，因此可以看到多个数值。直接从Github下载文件即可安装使用。
Github 项目地址：https://github.com/wahyd4/PM2. 5-Alfred

github项目地址：https://github.com/wahyd4/java-sorting
