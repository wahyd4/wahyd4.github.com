        也许我以前有的时候真的重色轻友，那是我还不懂。现在经过这么多事以后我才晓得，朋友至少和爱人（不是女朋友那么初级哈）同等重要，在某些方面甚至更重要。

        其实现在走进大学以后，真正更加觉得朋友的重要性。想想初中同学之间纯真的友谊，高中朋友间深厚的友谊，大学则是伙伴一样的友谊。每一样都难能可贵，我们需要的就是找到真正的朋友，那些关心你，不会出卖你的朋友。

有句话叫人生有一知己即可，而人生却需要很多的朋友，多多益善。每一条朋友都是一条路，这个世界便织成了一张大网。多么美丽的网。

        前几天和初中同学聚会，大家真的都成了帅小伙和靓妹了，一个二个都很不错，和你们聚在你们真的很幸福，那是久违幸福，三年多了，改变了太多，不会变的则是我们都是初零六级四班的学生。我们都是朋友。怀念那段简单的时光，也许我们以前都做得不够好，那就让我们以后多聚聚吧，亲爱的同学们，我们都要努力奋斗，我爱你们，希望明年的时候聚餐大家都来。

        高中，这三年改变了我的人生，虽然我这三年由于那个问题，让我三年都过得很痛苦，但是我还是相遇了我们伟大的25班，和最可爱的同学们。我们理科的娃娃就是乖，天天就是在教室里面学习，错过了好多加深友谊的机会哦。但是我们大家之间的友谊还是很淳朴的，感情还是很深厚的，我还记得我们高一时候的牛头山和那些什么什么牛哦。那段时光很快乐。还有足球队的兄弟们，虽然我脚法烂，但是我觉得我们还是多团结的，以后聚餐的时候一定要喝几杯。大家都在读书，都加油吧，五年后相信我们都是能干人。加油。

还有大学，软件工程的同学们，我们是相亲相爱的一家人哦，我们以后在一起的时间还很长，还有希望我们的足球队多踢球，培养哈感情塞。兄弟伙些，大家都是英雄。相信我们一定可以做出些成就把成大建设得更加美好（说大话了，哈哈）。

          还有我的小学的朋友们，还有那些通过网上认识的好朋友们。其实我觉得我们现在这个网络还是相对比较不那么虚假的，我真的通过这个网络认识很多朋友，恩，很到大家都聊得不错，我觉得只要有机会，一定要吧这些网络上的朋友变到现实中来，一直在网络上呆着没意思。当然美国的和台湾的当然希望不大。哈哈。谢谢所有的朋友们。

JAVA里面重要而基础的东西

发表于 2010-02-02 | 更新于: 2014-09-01

其实JAVA是在C和C++上开发的，因此他们很多东西是类似的。下面就是我学习的过程中觉得一些很重要的东西，写下来吧。

        
JAVA里面输出语句有两种。

        
System.out.print(“……”); 
这一个就相当于C里面的printf(“……”);

         System.out.println(“…….”);这一个就相当于printf(“……\n”);

        
第一个使用命令行参数程序

       
public class new1{

        
public static void main (String[] args){

         
System.out.print(“Hi, “);

          
System.out.print(args[0]);

           
System.out.println(“.How are you?”);

}

}

        
编译命令为   javac new1.java

                              
java     
new1   alice

                              
运行结果为   Hi, alice. How are
you?

   Integer.parseInt()是将String值转为int值。例如： int
a=Integer.parseInt(args[o]);

     
比如这个整数的乘法与除法。

  

 public  class new1

{

 public static void main(String[] args)

 {

  int
a=Integer.parseInt(args[0]);

  int
b=Integer.parseInt(args[1]);

  int sum=a+b;

  int prod=a*b;

  int quot=a/b;

  int rem=a%b;

  System.out.println(a+”+”+b+”=”+sum);

  System.out.println(a+”*”+b+”=”+prod);

  System.out.println(a+”/“+b+”=”+quot);

  System.out.println(a+”%”+b+”=”+rem);

  System.out.println(a+”=”+quot+”*”+b+”=”+rem);

 }运行结果为：

}

 

         
这里需要说明的就是，a+,+b+,+quot.

          
我们分析这句System.out.println(a+”%”+b+”=”+rem);

          
就可以发现
这个a+,+b+,+quot.类似于C语言里面的取地址。而且它是成对称分布的。如果只有一个变量的话则按最后的变量形式输出。例如：+quot;

          

         
\\\然后需要说的就是强制转换类型。其形式是比如：（int）1.10就是将1.10转化为整型

       
还有几个容易混淆的：

           
(int)11*0.3      
结果为：   3.3

           
11*(int)0.3      
结果为：  0

         
(int)(11*0.3)      
结果为：3

         
通过比较就可以看出区别了。

下面是强制转换整型的一个例子得到随机整数：

我看书上有一个数的自动提升，我大概懂。它有这样一句话，“你可以使用任何基本数值类型的数据，这些数据类型有用户希望的，很大的取值范围。因为Java自动将这些类型转换到更大的范围。”

同样有个例子。b,c
都是int类型的。  bb-4.0c这个系统会自动将b,c
变到double类型再进行计算，最后的结果是double类型的。如果你想得到int

 寒假之前就一直在说，这个寒假要怎么怎么样，要吧JAVA先学啊，但是计划总是赶不上变化，由于前几天出了点情况，导致整个人崩溃，现在才走出困境。所以我还是要尝试做我自己的事。那些啥子谈情说爱，我永远是被伤害的对象。好了，我还是会尽量写我的JAVA学习笔记。好了不说这么多了。开始吧。

    
   
要学JAVA当然要配置JAVA编译环境哦，首先要下载JDK。就到这个网址下面去下载就好了http://java.sun.com/javase/downloads/index.jsp具体到你是只下载JDK还是加上IDE编译环境就是你自己的事了。下载好厚当然是解压，安装。

    
   接下来就是配置环境变量了。这里我就说windows
7下面的方式，选择计算机然后右键属性，进去之后选择高级系统设置，在右下方就是环境变量。这里一共需要设置三个变量。没有的需要新建，有的就需要编辑就好。

  
   
 PATH：这个需要配置你安装JDK的路径到bin为止。例如：D:\Program
Files\Java\jdk1.6.0_17\bin

  
   
 JAVA_HOME：(注意这里是下划线，不是“-”)这个是需要配置到JDK安装环境就好。例如：D:\Program
Files\Java\jdk1.6.0_17

  
   
 CLASSPATH：这个配置就是“.”就是一个点哈，莫得引号。这个点表示在当前路径找。

  
   
这里的环境变量都是大写。配置好之后就是验证。

  
   
点击开始选项，在搜索框中输入cmd，调出DOS界面。在里面输入命令 javac
 或者java,如果显示出很多行代码，则表示配置成功，如果显示XX不是内部或外部命令，也不是可执行程序，则表示配置错误。

  
    接下来就是编译hello
world.

     public
static void main(String[] args) {

如果是用文本编辑器，将它保存到一个目录就好。接下来就是编译，在DOS里面找到这个路径。一般进去后默认是C盘用户路径，你就需要到你存文件的那个盘，比如我的是在D盘，然后我就输入
d:  然后回车 ，他就到D 盘了，然后就是在DOS里面复制地址，在windows 下复制地址，然后

HelloWorld.java,如果这一步回车没有提示表示编译成功，然后则是运行程序，java HeloWorld
（这里不需要文件名）。然后你就可以看到hello, world!(提示：JAVA里面是大写就是大写，比如这里面的System
String等都是首字母大写。要注意)
